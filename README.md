# Sign-up Form

Practice project from [The Odin Project](https://www.theodinproject.com/) 
— Intermediate HTML and CSS course.

## 🔍 Overview

A sign-up form built from scratch focusing on HTML structure, 
CSS styling, and client-side form validation.

## ✨ Features

- Responsive split-layout design
- Real-time input validation
- Semantic HTML structure

## 🧠 What I learned

- How to structure forms semantically with proper labels and inputs
- CSS custom properties for consistent theming
- Form validation using HTML attributes and CSS pseudo-classes (:valid, :invalid)
- Background-image techniques with cover and positioning

## 🛠️ Built with

- HTML5
- CSS3
